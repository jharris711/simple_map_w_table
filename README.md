# simple_map_w_table
Created with CodeSandbox


### Clone and change into directory:
```bash
git clone https://github.com/jharris711/simple_map_w_table.git && cd simple_map_w_table
```

### Install dependencies:
```bash
npm install
```

### Start dev server:
```bash 
npm start
```
